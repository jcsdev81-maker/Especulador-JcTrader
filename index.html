ESTRUTURA DE ONDAS
A contagem é o pilar principal. Toda leitura começa pela estrutura de ondas.
Impulso (1-2-3-4-5)
Onda 3 = mais forte. Onda 5 = final → possível reversão.
Correção (A-B-C)
Onda C = final da correção → entrada potencial.
Padrões
OCO, cunhas, triângulos (ABCDE), canais.

MACD 12/72/35
MACD Line (EMA12 - EMA72) + Signal Line (EMA35 do MACD) + Histograma (MACD - Signal). Config exata Bybit. Períodos configuráveis. 
//@version=5
indicator("MACD 12/72/35 [ESPECULADOR]", overlay=false)

// === INPUTS (config Bybit: 12, 72, Signal 35, MME) ===
len_fast = input.int(12, "Período Curto (EMA)", minval=1)
len_slow = input.int(72, "Período Lento (EMA)", minval=1)
len_signal = input.int(35, "Signal Length (EMA)", minval=1)
src = input.source(close, "Fonte")
col_macd = input.color(color.red, "Cor MACD Line")
col_signal = input.color(color.blue, "Cor Signal Line")
col_hist_up = input.color(color.new(#00E676, 30), "Histograma +")
col_hist_dn = input.color(color.new(#FF5252, 30), "Histograma -")

// === CÁLCULO ===
ema_fast = ta.ema(src, len_fast)
ema_slow = ta.ema(src, len_slow)
macd_line = ema_fast - ema_slow
signal_line = ta.ema(macd_line, len_signal)
histogram = macd_line - signal_line

// === PLOTS ===
plot(macd_line, "MACD Line", color=col_macd, linewidth=2)
plot(signal_line, "Signal Line", color=col_signal, linewidth=2)
plot(histogram, "Histograma", style=plot.style_histogram,
     color=histogram >= 0 ? col_hist_up : col_hist_dn, linewidth=2)
hline(0, "Zero", color=color.gray, linestyle=hline.style_dashed)

// === CRUZAMENTOS ===
bull_cross = ta.crossover(macd_line, signal_line)
bear_cross = ta.crossunder(macd_line, signal_line)
zero_bull = ta.crossover(macd_line, 0)
zero_bear = ta.crossunder(macd_line, 0)

DEMA 72
DEMA 72 como suporte/resistência dinâmico. Mesmo período da EMA lenta do MACD. Overlay no gráfico. 
//@version=5
indicator("DEMA 72 [ESPECULADOR]", overlay=true)

len = input.int(72, "Período DEMA", minval=1)
src = input.source(close, "Fonte")
col_bull = input.color(color.new(#00E676, 0), "Cor Bull (preço acima)")
col_bear = input.color(color.new(#FF5252, 0), "Cor Bear (preço abaixo)")

// === CÁLCULO DEMA ===
e1 = ta.ema(src, len)
e2 = ta.ema(e1, len)
dema_val = 2 * e1 - e2

// === PLOT — muda de cor conforme preço ===
dema_color = close > dema_val ? col_bull : col_bear
plot(dema_val, "DEMA 72", color=dema_color, linewidth=2)

// === ALERTAS ===
alertcondition(ta.crossover(close, dema_val), "Preço acima DEMA72", "Preço cruzou ACIMA da DEMA 72 — viés comprador")
alertcondition(ta.crossunder(close, dema_val), "Preço abaixo DEMA72", "Preço cruzou ABAIXO da DEMA 72 — viés vendedor")
